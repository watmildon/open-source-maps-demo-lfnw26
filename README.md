# Open Source Maps Demo — LinuxFest Northwest 2026

Demo maps built with MapLibre and the OpenStreetMap US data stack: [SliceOSM](https://slice.openstreetmap.us/), [Layercake](https://openstreetmap.us/our-work/layercake/), and [OpenTileService](https://tiles.openstreetmap.us/).

## Demos

- [Single Pin Conference Map](https://watmildon.github.io/open-source-maps-demo-lfnw26/single-pin-map.html) — Simple venue map replacing Leaflet + OSM Carto with MapLibre + OpenTileService
- [Better Conference Map](https://watmildon.github.io/open-source-maps-demo-lfnw26/lfnw_map.html) — Interactive map showing events during LinuxFest Northwest 2026
- [Bellingham Survey Needs](https://watmildon.github.io/open-source-maps-demo-lfnw26/) — Interactive map showing OSM features near parks and museums that need field surveying
